---
title: Quick Start
description: Create your first AllAgents workspace in minutes.
---

Get up and running with AllAgents in minutes.

## Create a Workspace

```bash
allagents workspace init my-workspace
cd my-workspace
```

## Add Plugins

```bash
allagents workspace plugin add code-review@claude-plugins-official
```

## Sync Plugins

```bash
allagents workspace sync
```

## Check Status

```bash
allagents workspace status
```
