# Getting Started

Welcome to YourProject! This guide will help you get up and running quickly.

## Prerequisites

- Node.js 18 or higher
- npm, yarn, or pnpm

## Installation

```bash
npm install -g yourproject
```

## Quick Start

1. Initialize a new project:

```bash
yourproject init my-app
```

2. Navigate to your project:

```bash
cd my-app
```

3. Start the development server:

```bash
npm run dev
```
